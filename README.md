# PEWMA Clinic Home 🏥✨

> **"Cuidamos tu salud donde más importa: tu hogar."**

Sitio web oficial de **PEWMA Clinic Home**, una plataforma web diseñada para presentar y facilitar el acceso a servicios profesionales de enfermería domiciliaria integral dentro de la **Región Metropolitana, Chile**. El sitio web funciona como una Landing Page informativa y canal de contacto directo para pacientes y sus familias.

## 🚀 Características del Sitio Web

- **Diseño Responsivo e Inclusivo:** Optimizado para dispositivos móviles, tablets y ordenadores mediante media queries fluidas (`clamp`) y soporte de accesibilidad básico (`aria-labels`, `role="list"`, `sr-only`).
- **Navegación Fluida:** Menú con comportamiento de desplazamiento suave (`scroll-behavior: smooth`) anclado a las secciones clave.
- **Canal de Atención Directo:** Integración estratégica con WhatsApp para agilizar la solicitud de atención clínica domiciliaria.
- **Optimizado para el Rendimiento:** Carga optimizada mediante el uso de fuentes web externas (`DM Sans`, `DM Serif Display`) con pre-conexión y gráficos vectoriales nativos (SVG).

## 🛠️ Servicios Clínicos Expuestos

El sitio detalla un catálogo integral de prestaciones de enfermería ejecutadas por personal certificado (enfermeras/os titulados con registro y experiencia clínica):

1. **Toma de exámenes domiciliarios:** Extracción de muestras de sangre, orina y otros exámenes de laboratorio.
2. **Manejo de dispositivos invasivos:** Instalación y cuidado de traqueotomía, gastrostomía, sonda nasogástrica, sonda Foley, ostomías y drenajes.
3. **Curaciones simples y avanzadas:** Tratamiento de heridas agudas y crónicas, úlceras por presión, heridas quirúrgicas y retiro de puntos.
4. **Educación en salud:** Capacitación familiar y al paciente en el manejo de dispositivos y administración de fármacos.
5. **Administración de medicamentos:** Aplicación endovenosa, intramuscular y subcutánea bajo técnica estéril y protocolos seguros.
6. **Administración de anticonceptivos:** Inyectables con seguimiento personalizado y orientación profesional.

## ⚙️ Tecnologías y Arquitectura

El proyecto está construido como un sitio estático monolítico, ideal para un despliegue rápido y eficiente en plataformas como **GitHub Pages**:

*   **HTML5 Semántico:** Uso de etiquetas estructurales (`<nav>`, `<section>`, `<article>`, `<footer>`) para mejorar el SEO y la accesibilidad.
*   **CSS3 Custom (Vanilla):** Estilizado adaptado sin dependencias de frameworks externos. Incluye una paleta de colores corporativa basada en tonos *Aqua Teal* (`#1dba8e`, `#0e8f6b`, `#d0f4ea`) y soporte para accesibilidad de animaciones (`prefers-reduced-motion`).
*   **Componente de Menú Móvil:** Control interactivo de despliegue mediante manipulación de clases nativas en JavaScript.
